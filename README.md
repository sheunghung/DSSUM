# DSSUM
The data for paper **Towards Speaker-Aware Dialogue Speech Summarization**

## Dataset

DiaSPEECH is a large-scale spoken dialogue summarization dataset built on DIALOGSUM, comprising 13,460 dialogues with speaker-aware abstractive summaries.

We are releasing the data in stages:
Validation and test sets (audio, transcripts, summaries): available now
Training set: released upon paper acceptance
Code: released after cleanup
